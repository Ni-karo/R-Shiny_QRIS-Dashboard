# QRIS Distribution in Indonesia  
[![ShinyApps.io](https://img.shields.io/badge/ShinyApp-Live%20Demo-blue)](https://nindya-karona.shinyapps.io/Dashboard-QRIS-Distribution-in-Indonesia/)

## Overview  
This interactive dashboard visualizes **QRIS (Quick Response Code Indonesian Standard) adoption and payment behavior** in Indonesia. It provides insights into the distribution of payment methods across different age groups, spending behavior, and user sentiments.

## Features  
✅ **Interactive Data Visualization** – Dynamic plots with filtering options  
✅ **Demographic Analysis** – Breakdown by age groups and payment methods  
✅ **Sentiment & Thematic Analysis** – Insights from qualitative survey responses  
✅ **User-Friendly Dashboard** – Accessible via web, no installation needed  

## 🚀 Live Demo  
🔗 **[Explore the Dashboard](https://nindya-karona.shinyapps.io/Dashboard-QRIS-Distribution-in-Indonesia/)**  

## 🛠️ Languange Used  
- **R Shiny** – For building the interactive web app  
- **ggplot2 & ggiraph** – For interactive visualizations  
- **tidyverse** – For data processing  
- **textdata & tidytext** – For sentiment analysis  
- **readxl & openxlsx** – For handling Excel files  

## 📂 Data Sources  
The dataset used in this dashboard is based on a **survey of digital payment behavior in Indonesia**, which includes:  
- **Demographic data** (Age, Payment Method, Expenditure, etc.)  
- **Transaction patterns** (Frequency of QRIS usage)  
- **Open-ended responses** for sentiment and thematic analysis
